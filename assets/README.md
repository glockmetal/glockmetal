# Alimony Adventure - Art Assets

This folder contains all art assets for the game. Drop your images into the appropriate subfolders.

## Folder Structure

```
assets/
├── characters/       # All character art
│   ├── player/       # Player character sprites/portraits
│   ├── child/        # Your child's portraits (different ages)
│   ├── npcs/         # Friendly NPCs (fellow workers, etc.)
│   └── enemies/      # Enemy sprites for each job type
│
├── ui/               # User interface elements
│   ├── menus/        # Menu backgrounds, frames
│   ├── icons/        # Small icons (health, money, etc.)
│   ├── buttons/      # Button graphics
│   └── hud/          # In-game HUD elements
│
├── screens/          # Full screen backgrounds
│   ├── title/        # Title screen art
│   ├── intro/        # Intro sequence scenes (1-6)
│   ├── gameover/     # Game over screen art
│   └── victory/      # Victory screen art
│
├── jobs/             # Job-specific environment art
│   ├── fastfood/     # Fast food restaurant
│   ├── warehouse/    # Warehouse environment
│   ├── rideshare/    # Car/street scenes
│   ├── construction/ # Construction site
│   ├── security/     # Security booth/building
│   ├── callcenter/   # Office cubicles
│   ├── delivery/     # Delivery scenes
│   ├── office/       # Office environment
│   └── hospital/     # Hospital environment
│
├── items/            # Item sprites
│   ├── weapons/      # Weapon icons
│   ├── armor/        # Armor icons
│   ├── consumables/  # Food, drinks, healing items
│   └── money/        # Cash, coins, tips
│
├── effects/          # Visual effects
│                     # (damage flash, level up, etc.)
│
└── backgrounds/      # Tile sets and backgrounds
```

## Recommended Image Formats
- **PNG** - Best for sprites with transparency
- **JPG** - Good for backgrounds without transparency
- **SVG** - Scalable icons (optional)

## Naming Convention
Use lowercase with hyphens:
- `player-idle.png`
- `player-walk-1.png`
- `enemy-karen.png`
- `item-coffee.png`
