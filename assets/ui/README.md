# UI Elements

User interface graphics.

## menus/
- `menu-background.png` - Main menu background
- `menu-frame.png` - Border/frame for menus
- `selector.png` - Menu selection indicator

## icons/
Small status icons (16x16 or 32x32 recommended)
- `heart.png` - Health
- `dollar.png` - Money
- `clock.png` - Time/days remaining
- `child-icon.png` - Child age indicator
- `level.png` - Player level
- `attack.png` - Attack stat
- `defense.png` - Defense stat

## buttons/
- `button-normal.png` - Default button state
- `button-hover.png` - Hovered button
- `button-pressed.png` - Clicked button
- `dpad-*.png` - Mobile D-pad buttons

## hud/
In-game interface elements
- `health-bar.png` - HP bar frame
- `health-fill.png` - HP bar fill
- `money-display.png` - Money counter frame
- `log-background.png` - Message log background
- `minimap-frame.png` - If adding minimap
